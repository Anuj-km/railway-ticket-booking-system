# 🚆 Railway Ticket Booking System

A web-based railway ticket booking system built using PHP and MySQL.  
It supports booking, cancellation, and viewing ticket status with basic allocation logic.

---

## ✨ Features
- Book tickets for multiple passengers
- Ticket allocation system (Confirmed / Waiting)
- Cancel tickets
- View booking details

---

## 🛠 Tech Stack
- PHP
- MySQL
- HTML/CSS

---

## ⚙️ How to Run
1. Install XAMPP
2. Move project folder to:
   C:\xampp\htdocs\
3. Start Apache and MySQL
4. Import the database file into phpMyAdmin
5. Open browser:
   http://localhost/railway
